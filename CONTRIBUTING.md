# Contributing to Jarvis Research Showcase

Thank you for your interest in the Jarvis project!

## How to Contribute

### 1. Ask Questions
Open an issue with the `question` label if something is unclear.

### 2. Share Ideas
Have a suggestion? Open an issue with the `idea` label.

### 3. Report Issues
Found a problem in the documentation? Open an issue with the `bug` label.

### 4. Improve Documentation
- Fork the repository
- Make your changes
- Submit a pull request

## Code of Conduct

- Be respectful and constructive
- Focus on ideas, not people
- Help others learn
